5)布局   
发展：
Web设计1.0是“一维的”：设计元素大多是按顺序排列的（按文档流的自然顺序排列）   

Web设计2.0是“二维的”：单元格中有放置元素的网格，具有更多的自由性     

Web设计3.0是一个“新的维度”：它可以像平面设计工具一样的自由地定位元素、重叠。为网页设计开辟了新的前景。也将开启新的Web页面设计时代
